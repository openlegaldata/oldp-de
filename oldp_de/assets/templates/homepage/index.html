API

Unsere REST API ermöglicht es dir alle Inhalte in maschinenlesbarer Form abzurufen. In der Regel können alle API-Anfragen (z.b. Abrufen von Urteilen) ohne Anmeldung getätigt werden, ansonsten ist ein API-Schlüssel notwendig. Um einen API-Schlüssel zu erhalten musst du dir einen Account registrieren und dich einloggen. Über verschiedene Endpoints kann auf Urteile, Gesetze u.v.m zugegriffen werden. Um die API nutzen zu können, bieten wir eine Reihe von API-Clients in Python, PHP, Java und Javascript an. Die ausführliche Dokumentation ist ein guter Start um sich mit den Features der API vertraut zu machen. Zusätzlich möchten wir dir unsere Beispiele ans Herz legen.

Fragen? Kommentare? Was für coole Sachen hast du mit der API gebaut? Schreib uns.

Mitmachen

Dieses Projekt wird von Freiwilligen ehrenamtlich in der Freizeit betrieben. Deshalb sind wir auf Deine Unterstützung angewiesen. Wir suchen ständig Aktivisten, Entwickler, Juristen, Journalisten, Analysten und vor allem motivierte Mitstreiter*innen. Du hast Lust den Rechtstaat gemeinsam mit uns und offnen Daten etwas transparenter zu machen? Dann schreibe uns eine Nachricht, sende uns Gerichtsentscheidungen, nutze unsere Daten, melde Bugs oder arbeite mit an neuen Features.

Der Quellcode der Plattform und ein Python SDK ist bereits unter freier MIT Lizenz auf GitHub veröffentlicht. Also - nicht nur zu gucken, sondern Mitmachen!

Über das Projekt

Open Legal Data ist ein nicht-gewinnorientiertes Projekt zur Förderung des freien Zugang zu juristischen Daten. Unser Hauptanliegen ist es technische Mittel zur Verarbeitung und Analyse von juristischen Inhalten, insbesondere Rechtsprechung und Rechtsnormen, zur Verfügung zu stellen. Durch das Zugänglichmachen mit Hilfe von APIs oder das Veröffentlichen von Datensätzen soll der Umgang mit Rechtsthemen für juristische Laien erleichtert werden. Mit unserer Plattform richten wir uns in erster Linie an technisch versiert AktivistenInnen, JournalistenInnen und WissenschaftlerInnen.

Open Legal Data verpflichtet sich mit diesem Projekt den Prinzipien der Open Data Bewegung und der Erklärung des Free Access To Law Movements. Auf unserer englischen Seite findet ihr weitere Informationen über das Projekt.
